Project About

This project is the development of a mobile application. It was created to learn how to clone existing applications. The mobile application is developed using the Dart programming language and the Flutter framework.

In this application, the concept of "SIWASLU Clone" refers to a cloned application of an existing app called SIWASLU. This application is used for data recap and reporting of the 2024 election results, including reporting for the presidential election, the House of Representatives (DPR RI), regional parliament (DPRD), and others. The reported results will be sent to the provided server; however, this is only a clone of an already existing and functioning application.
